Pearson Education Case Study: Success Story

API Gateway for the education sector: parental control

Location: Warsaw, Poland Industry: Education

Pearson Central Europe has trusted KrakenD as its API Gateway solution for the Parent App, which provides parents with a wide range of resources, including podcasts, infographics, quizzes, and checklists, to support their children’s progress in learning.

KrakenD API Gateway offers the endpoints to the Parent App and sits between end-users and the Pearson services, which provide all the data that KrakenD aggregates to supply the use case.

KrakenD handles OAuth with HMAC security and implements minor custom middlewares to fit the desired end-user consumption of the services.

Pearson reported that KrakenD has been running since the day it was in production (2018) without a single issue.
