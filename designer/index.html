KrakenDesigner

An UI to edit the KrakenD configuration file

Open the KrakenDesigner

What is the designer?

KrakenDesigner allows you to edit the KrakenD configuration using your browser. It is a simple javascript application that runs in your browser without sending any data to the server. If you want to use it offline you can download the source, if not you can always use our up to date link.

Clone KrakenDesigner in Github

Or use the Docker image:

docker run --rm -p 8080:80 devopsfaith/krakendesigner
