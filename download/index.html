Docker / Kubernetes

  • Docker Image for Production

    Run anywhere. The official KrakenD image ready for production usage

    Consider copying the configuration in your immutable image instead of mounting a volume.

    docker run -p 8080:8080 -v $PWD:/etc/krakend/ krakend:2.12.0

  • Docker Image for Development

    The KrakenD image with hot-reload behaviour, for non-production usage

    To develop locally with KrakenD, pull this image on Windows, Mac, or Linux. It reloads the configuration automatically when you save changes, to save time testing changes.

    docker run -p 8080:8080 -v $PWD:/etc/krakend/ krakend:watch

    * Requires building the tag first - read more

On-Prem / Packages

  • Ubuntu or Debian

    DEB packages

    To install KrakenD on Ubuntu, Debian, or other distros with DEB packages, paste the following lines in the terminal using a privileged user:

    # sudo su
apt install -y ca-certificates gnupg
apt-key adv --keyserver keyserver.ubuntu.com --recv 5DE6FD698AD6FDD2
echo "deb https://repo.krakend.io/apt stable main" \
    | tee /etc/apt/sources.list.d/krakend.list
apt-get update
apt-get install -y krakend

  • Redhat, CentOS, RockyLinux

    RPM packages

    To install KrakenD using the RPM package, paste the following lines in the terminal using a privileged user:

    rpm -Uvh https://repo.krakend.io/rpm/krakend-repo-0.2-0.x86_64.rpm
yum install -y krakend
systemctl start krakend

  • Fedora

    Installation with dnf

    To install KrakenD on Fedora using dnf run the following commands:

    rpm -Uvh https://repo.krakend.io/rpm/krakend-repo-0.2-0.x86_64.rpm
dnf install -y krakend
systemctl start krakend

  • Linux generic

    AMD64 / ARM64

    You can run Krakend on any GNU/Linux system by running the binary included in the following tar.gz file. Download and decompress anywhere.

    AMD64: Download tgz SHA | PGP
    ARM64: Download tgz SHA | PGP
    How to check the SHA and PGP signature

  • Homebrew

    Mac OS X installation

    If you have Homebrew installed, the following command downloads, compiles and installs KrakenD from source:

    brew install krakend
