Install KrakenD using Homebrew

Paste the following lines in the terminal. You'll need Homebrew installed:

brew install krakend

The process might take a while as it compiles the binary for your platform

Install KrakenD with Debian package

Paste the following lines in the terminal:

apt-key adv --keyserver keyserver.ubuntu.com --recv AB39BEA1
echo "deb http://repo.krakend.io/apt stable main" | tee /etc/apt/sources.list.d/krakend.list
apt-get update
apt-get install -y krakend

What these commands are doing:

  • Add the key
  • Add the repo to the sources.list
  • Update your package list
  • Install the KrakenD service

The process has been tested on Debian 8, Debian 9 and Ubuntu 16.x

Install KrakenD with RPM package

Paste the following lines in the terminal:

rpm -Uvh http://repo.krakend.io/rpm/krakend-repo-0.1-0.noarch.rpm
yum install -y krakend
systemctl start krakend

What these commands are doing:

  • Install the repo package
  • Install the KrakenD package
  • Start the KrakenD service

Fedora only

Since fedora has deprecated yum in favor of dnf, you must replace one for the other (btw, dnf will start your KrakenD service):

rpm -Uvh http://repo.krakend.io/rpm/krakend-repo-0.1-0.noarch.rpm
dnf install -y krakend
systemctl start krakend

The process has been tested on Centos 7 and Fedora 24
