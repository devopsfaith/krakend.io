KrakenD 2.3.1 (patch version) released

by Albert Lombarte

Apr 26, 2023

1 min read

A new patch version KrakenD 2.3.1 is available to download page. Upgrading from any 2.x is backwards compatible.

🚀 Summary of changes for CEv2.3

The new TLS Client functionality was not included in the release.

  • The deny attribute did not work on the third nesting level and above.
  • The new client_tls attribute was not yet included.

Upgrading to the latest version is always advised.
